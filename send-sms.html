📱 알리고 SMS 발송

관리자 전용 - 공지사항 문자 발송

⚠️ 주의사항
  • 이 페이지는 관리자만 사용해야 합니다
  • SMS 발송 시 비용이 발생합니다 (건당 약 15~20원)
  • API 키를 절대 다른 사람과 공유하지 마세요

📋 사용 방법

1 알리고에서 API 키와 발신번호를 확인하세요
2 아래 폼에 정보를 입력하세요
3 "회원 목록 불러오기" 버튼을 클릭하세요
4 문자 내용을 작성하고 "SMS 발송" 버튼을 클릭하세요

🔑 알리고 설정

알리고에서 등록한 발신번호를 입력하세요

👥 수신 대상

위 버튼을 클릭하여 회원 목록을 불러오세요

✉️ 문자 내용

0 / 90자 (단문 SMS)
💡 팁:
  • 90자 이하: 단문 SMS (15~20원)
  • 91자 이상: 장문 LMS (40~50원)
  • 짧고 명확하게 작성하세요
